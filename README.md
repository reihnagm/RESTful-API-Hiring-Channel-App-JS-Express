## Installation

### Clone

```bash
$ git clone https://github.com/reihnagm/simple-restful-api
$ cd simple-resful-api
$ npm install
```

### Create Environment Variable
```bash
$ cp .env.example .env
$ nano .env
```

---
### Start Development Server
```bash
$ npm start
```
